**SecureVault** is an elite encryption suite designed for those who demand ultimate privacy without compromising on design. Built with high-performance **ChaCha20-Poly1305** and **Argon2id**, SecureVault provides a virtually impenetrable fortress for your files, passwords, and private notes.

## ✨ Features

### 📂 File & Folder Locker
Transform any sensitive data into encrypted `.vlt` containers. SecureVault handles multi-gigabyte directories with multi-threaded parallel processing for lightning-fast security.

### 🔑 Password Manager
A sleek, encrypted vault for your digital life. Featuring intelligent sorting, secure generation, and military-grade persistence.

### 📝 Secure Notes
A distraction-free, glassmorphism-styled editor designed for your private thoughts. Every character is encrypted before it ever touches your disk.

### 🚜 Secure Shredder
Wipe your tracks forever. Implementing military-grade deletion standards (DoD 5220.22-M) to ensure your data is unrecoverable by any forensic tool.

### 💥 Anti-Forensics & Self-Destruct
Automated vault destruction after multiple failed access attempts. Your security scales with the threat.

## 🚀 Getting Started

### 📋 Prerequisites
* **OS**: Windows 10/11 (Optimized for native integrations)
* **Environment**: Python 3.14+ (If running from source)

### 🛠️ Installation

1. **Download the Executable**:  
   Visit the [Releases](https://github.com/sudosilva/SecureVault/releases) page and download `SecureVault.exe`.

2. **Run from Source**:  
   If you prefer to build it yourself:
   ```bash
   git clone https://github.com/sudosilva/SecureVault.git
   cd SecureVault
   pip install -r requirements.txt
   python main.py gui
   ```
